GitHub User Search

Welcome to GitHub User Search!

This application allows you to search for GitHub users and view their profile details and repositories. Simply enter a GitHub username in the search bar below and click the "Search" button.

On the "User Details" tab, you will see information about the user, including their username, name, bio, number of public repositories, followers, and following count. The "Repositories" tab will display a list of their repositories, with pagination support to navigate through all repositories.
